Description

A multi select drop down list directive for AngularJS. This directive takes an array of value/label objects and formats them as a drop down list where multiple options can be selected. It also has shortcut filters to select all and select none, and an optional text box to enter new values. The element returns a comma separated string of the selected options.

This is a demo Yeoman generated project containing the directive with accompanying view and styles.

Installation

To get the demo project running, use the following commands:

$ git clone https://github.com/shalotelli/angular-multiselect.git multiselect
$ cd multiselect
$ npm install
$ bower install

To start the project with grunt, do:

$ grunt serve

Usage

To use this directive, call use the multi-select tag, including the model with the array of data objects and any display options (listed below).

<multi-select
  values="values"
  show-filters="true"
  show-other="true">
</multi-select>

Demo

Options

Option Values Description Default Value
show-filters true/false Show select all/select none true
show-other true/false Enable user to enter custom values false
